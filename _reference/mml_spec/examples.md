# MML 入力と期待イベント例

この文書は `ctrmml/src/unittest/test_mml_input.cpp` を元に、Rust 実装の回帰テストへ転用しやすい形で代表ケースをまとめたものです。

## 1. 基本ノート列

入力:

```text
A cdef
```

期待:

- track A に 4 イベント追加
- すべて `NOTE`

## 2. オクターブ

入力:

```text
A o4cd>e<f
```

期待ノート番号:

- `c` → `36`
- `d` → `38`
- `>e` → `52`
- `<f` → `41`

## 3. 臨時記号

入力:

```text
A o4c+d-
```

期待ノート番号:

- `c+` → `37`
- `d-` → `37`

## 4. 長さと付点

入力:

```text
A l16cd8e.f8.
```

期待 `on_time`:

- `c` → `6`
- `d8` → `12`
- `e.` → `9`
- `f8.` → `18`

前提:

- 全音符長は既定の `96`
- 量子化は既定の `Q8`

## 5. 全音符長変更

入力:

```text
A C96 c1 c2 c4 C128 c1 c2 c4 C192 c1 c2 c4
```

期待 `on_time`:

- `C96`: `96`, `48`, `24`
- `C128`: `128`, `64`, `32`
- `C192`: `192`, `96`, `48`

## 6. ループ

入力:

```text
A [c] [c]5
```

期待イベント列:

1. `LOOP_START`
2. `NOTE`
3. `LOOP_END param=2`
4. `LOOP_START`
5. `NOTE`
6. `LOOP_END param=5`

## 7. 複数トラック複製

入力:

```text
ABC cdef
```

期待:

- track A, B, C それぞれに 4 ノートずつ追加

## 8. 条件ブロック

入力:

```text
ABC o4c{d/e/f}g
```

期待:

- 3 トラックともイベント数は 3
- 先頭 `c` は全トラック `36`
- 終端 `g` は全トラック `43`
- 中央ノートだけ分岐
  - A: `38`
  - B: `40`
  - C: `41`

## 9. キーシグネチャ

入力:

```text
A _{c+} l4 o4cdefgab>c
A _{=dg} l4 o4cdefgab>c
A _{C} l4 o4cdefgab>c
```

期待:

### `_{c+}` の結果

- `c+ d+ e f+ g+ a b c+`
- ノート番号: `37, 39, 40, 42, 44, 45, 47, 49`

### `_{=dg}` の結果

- `d` と `g` だけナチュラル化
- ノート番号: `37, 38, 40, 42, 43, 45, 47, 49`

### `_{C}` の結果

- C major へリセット
- ノート番号: `36, 38, 40, 41, 43, 45, 47, 48`

## 10. プラットフォーム固有メッセージ

入力:

```text
A 'first second third'c 'foo bar baz'
```

期待:

- イベント数は 3
- 1 個目と 3 個目は `PLATFORM`
- platform command tag に次の値が入る
  - `first second third`
  - `foo bar baz`

## 11. echo

入力:

```text
A \=2,3 l4 o4c4\8 \=-1,3d\r\ \=-4,0>e\< \=-2,2f\ \=2,2f\
```

期待される重要点:

- 最初の echo は buffer 不足のため休符
- 負の delay は buffer を保持
- volume 差分がある場合は前後に `VOL_REL` が入る
- volume が `0` の場合は `VOL_REL` が入らない
- 正の delay 指定は buffer をクリアする

## 12. 失敗ケース

入力:

```text
A l0cdef
A l-2cdef
A l:-5cdef
```

期待:

- いずれも parse error

## Rust 実装用の最小回帰セット

最初の段階では、次の順でテスト化するのが現実的です。

1. `o`, `<`, `>`, 音符番号
2. `l`, `C`, 付点, `:ticks`
3. `^`, `&`, `R`, `~`
4. `_{}` キーシグネチャ
5. `[]`, `{}` 条件ブロック
6. `\=` / `\`
