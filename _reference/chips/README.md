# PSG チップ参照メモ

このディレクトリは、`SN76489` と `AY-3-8910` の実装差分を段階的に整理するための参照メモ置き場です。ここに置く文書は、実装方針・比較観点・法的メモをまとめるためのものであり、外部実装のコード断片をそのまま転載することは意図していません。

## 運用方針

- まず `SN76489` を基準モデルとして period / volume / noise の扱いを整理する
- `AY-3-8910` は共通 PSG 骨格の上に、振幅テーブル、noise、mixer、hardware envelope を段階追加する
- MAME など既存エミュレータは「挙動確認の参照元」として扱い、実装コードは独自に記述する
- データシートや一次資料を優先し、二次資料は差分確認や補助説明に限定する
